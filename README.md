# OS Homework – Process Scheduling Simulation

## Overview
This project simulates CPU scheduling algorithms and memory management in an operating system.

## Implemented Algorithms
- First-Come, First-Served (FCFS)
- Shortest Job First (SJF - Non-Preemptive)
- Round Robin (RR)

## Features
- Gantt chart representation of process execution
- Waiting Time (WT) and Turnaround Time (TAT) calculation
- CPU Utilization calculation
- First-Fit Memory Allocation (Bonus)

## Files Included
- scheduler.cpp – Source code
- processes.txt – Input file
- output_FCFS.txt – FCFS results
- output_SJF.txt – SJF results
- output_RR.txt – Round Robin results
- OS_Project_Report.pdf – Project report
